\# FC Stat Checker



A RuneLite plugin for Friends Chat hosts to quickly check if members in their FC meet a minimum skill level requirement.



\## How to use

1\. Join a Friends Chat in game

2\. Open the FC Stat Checker panel from the side toolbar

3\. Select the skill you want to check from the dropdown

4\. Set the minimum level required

5\. Optionally toggle "Unranked only" to only show members with no hiscore entry for that skill

6\. Click "Search FC"

7\. Anyone who does not meet the requirement will appear in the results list



\## Results

\- Red = below the minimum level requirement

\- Amber = unranked on the hiscores for that skill

\- Gray = player not found on hiscores



\## Notes

\- The search checks the OSRS hiscores API for each member one at a time

\- There is a short delay between each lookup to avoid hitting the API too hard

\- Only players who fail the requirement are shown in the results
